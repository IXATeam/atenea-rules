■ SINTAXIS C# OBLIGATORIA:
- NUNCA uses LINQ (.Where, .Cast, .Select, .First, .ToList, .Any)
- NUNCA uses interpolacion de strings ($"texto {var}") — usa: "texto " + var
- NUNCA uses lambdas (=>) excepto en Sort((a,b) => ...)
- USA foreach en lugar de LINQ para iterar colecciones
- Las variables 'document' y 'uidoc' ya estan disponibles — NO las redeclares
- ElementId requiere sufijo L: new ElementId(12345L)
- Para obtener valor numerico de ElementId usa: id.Value (NO .IntegerValue)
- NUNCA usar Transaction directa — usa SubTransaction
- Un solo bloque de codigo por respuesta — nunca dos bloques separados

■ UNIDADES:
- Revit usa pies internamente: metros * 3.28084 = pies
- Para mostrar: pies * 304.8 = mm | pies * 0.3048 = metros
- XYZ siempre en pies: new XYZ(x_pies, y_pies, z_pies)

■ NIVELES:
Level nivel = null;
FilteredElementCollector col = new FilteredElementCollector(document);
col.OfClass(typeof(Level));
foreach (Level l in col) { if (l.Name == "Nivel 1") { nivel = l; break; } }
if (nivel == null) { foreach (Level l in col) { nivel = l; break; } }

■ REJILLAS (GRID):
- Lineas SIEMPRE en plano XY con Z=0
- Eje vertical (Y): Line.CreateBound(new XYZ(x,-30,0), new XYZ(x,30,0))
- Eje horizontal (X): Line.CreateBound(new XYZ(-30,y,0), new XYZ(30,y,0))
- Grid.Create(document, line) — no necesita nivel
- Grid.Curve devuelve Curve (NO CurveArray)
- NUNCA uses grid.Elevation — no existe
- Interseccion de grids:
  IntersectionResultArray ira;
  SetComparisonResult scr = gridA.Curve.Intersect(grid1.Curve, out ira);
  XYZ punto = (scr != SetComparisonResult.Disjoint && ira != null && ira.Size > 0)
      ? ira.get_Item(0).XYZPoint
      : new XYZ(gridA.Curve.GetEndPoint(0).X, grid1.Curve.GetEndPoint(0).Y, 0);

Wall.Create(document, curve, wallTypeId, levelId, height, offset, flip, structural)
